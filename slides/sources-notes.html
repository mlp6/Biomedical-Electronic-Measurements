Superposition

Superposition Statement

  • In a linear circuit containing \(N\) sources, each branch voltage and current is the linear sum of \(N\) voltages and currents (i.e., each source contributes to part of the total element current and voltage).

  • Each source’s contribution can be calculated by “eliminating” the contribution of the other sources and solving for element currents/voltages.

“Eliminating” Sources

  • “Eliminating” a source means removing its voltage or current contribution to the circuit.

  • A voltage sourse can be eliminated by replacing it with a short circuit.

    • Branch current can still be conducted.

    • Forcing no \(\Delta\) V related to the voltage source.

  • A current source can be eliminated by replacing it with an open circuit.

    • Voltage difference allowed between two nodes.

    • Blocking current from flowing from other sources because the eliminated current source exclusively dictactes that branch current.

Superposition Example

Note

Solve for \(i_3\) using source superposition.

  • \(V_S = 10 V\)

  • \(R_1 = 200 \Omega\)

  • \(R_2 = 200 \Omega\)

  • \(R_3 = 100 \Omega\)

  • \(I_S = 0.1 A\)

Note

\[ i_3 = i_{3_{V_S}} + i_{3_{I_S}} \]

Step 1: Eliminate \(I_S\); Solve for \(i_{3_{V_S}}\)

\[ i_{3_{V_S}} = \frac{V_S}{R_1+R_2+R_3} = 0.2 A \]

Note

\(i_{3_{V_S}}\) is the contribution to total \(i_3\) from \(V_S\).

Step 2: Eliminate \(V_S\); Solve for \(i_{3_{I_S}}\)

Tip

While any circuit analysis techinque we have used to date is fair game, current division resonates with me…

\[ I_{3_{I_S}} = \frac{v_A}{R_2+R_3} = -I_S\left(\frac{R_1}{R_1+R_2+R_3}\right) = -0.04 A \]

Step 3: Sum Contributions from \(I_S\) and \(V_S\)

\[ i_3 = i_{3_{V_S}} + i_{3_{I_S}} = 0.02 A + (-0.04 A) = -0.02 A \]

Equivalent Circuits

Important

Each “block” (physical or functional) of a circuit can be represented by a 2-terminal equivalent circuit, composed of a single source and resistance that completely captures the \(i-v\) characteristics of that block.

  • If that single source is a voltage source, then it is known as a Thevenin source equivalent.

  • If that single source is a current source, then it is known as a Norton source equivalent.

Thevenin Equivalent Sources

Thevenin Theorem

When a circuit is viewed from the perspective of the load, any source network composed of ideal voltage/current sources and linear resistors (i.e., defined by Ohm’s law), can be replaced by an equivalent circuit with:

  1. An ideal voltage source (\(V_T\)), in series with

  2. An equivalent resistance (\(R_T\)).

How do we solve for \(V_T\) and \(R_T\)?

Step 1: Find the equivalent resistance

Find the equivalent resistance as “seen” by the terminals of that source block.

  1. Remove the load from the terminals of the block.

  2. Eliminate all independent voltage and current sources.

Note

To date, we have only dealth with independent voltage and current sources (i.e., their source voltage or current values are not dependent on other voltages/currents in the circuit).

  1. Compute the equivalent resistance with the load removed.

Step 2: Find the equivalent source voltage

Solve for the equivalent Thevenin source voltage, which is equal to the terminal voltage when the terminals are “open” (i.e., no load is attached to them).

  1. Remove the load from the terminals (create an “open” load condition).

  2. Keep all sources.

  3. Solve for \(V_T = V_{{ab}_{OC}}\)

Why use a Thevenin Equivalent Circuit?

Important

A major benefit of using an equivalent source is that if we have an unknown or changing load (\(R_L\)), we can easily solve for \(i_L\) as a simple voltage division with known \(V_T\) and \(R_T\).

Extremely useful for instrument design!

  • Current draw as a function of load (e.g., make sure to not exceed \(I_{max}\) of a supply).

  • Power consumption as a function of load.

Norton Equivalent Sources

Norton Theorem

When a circuit is viewed from the perspective of the load, any source network composed of ideal voltage/current sources and linear resistors (i.e., defined by Ohm’s law), can be replaced by an equivalent circuit with:

  1. An ideal current source (\(V_T\)), in parallel with

  2. An equivalent resistance (\(R_N = R_T\)).

Step 1: Calculate \(R_N\)

Important

This is the same process as calculating the Thevenin equivalent resistance (\(R_T\))!

Step 2: Calculate the Norton Equivalent Source

  1. Similar to calculating the Thevenin equivalent source, but instead of solving for the open-circuit voltage at the terminals, solve for the short circuit current at the terminals.

  2. This short circuit current is the Norton source current (\(I_N\)).

Proof that Thevening & Norton Equivalents Circuits are Equivalent

Note

The circuits are equivalent if they have the same \(i-v\) characteristics.

Thevenin

\[ \begin{gather} V_L = V_T \frac{R_L}{R_T+R_L} \\ i_L = \frac{v_L}{R_L} = \frac{V_T}{R_T+R_L} \end{gather} \]

Norton \[ \begin{gather} i_L = i_N \frac{R_T}{R_N+R_L} \\ V_L = i_L R_L = I_N R_N \frac{R_L}{R_N+R_L} \\ i_N = \frac{V_T}{R_T} \\ V_L = V_T \frac{R_L}{R_T+R_L} \end{gather} \]

They are the same!

Source Transformation

Note

We can use these equivalent circuits to transform between voltage and current sources using Ohm’s law:

\[ V_T = I_N R_T \]

Tip

Use source transformations to change a circuit into a voltage or current division problem that is easier to solve!
